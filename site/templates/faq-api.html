How can I automate interactions with the Janitor?

There is a JSON API that can be used to retrieve information about packages, to reschedule runs and trigger creation of merge proposals.

The API provides endpoints for:

  • Retrieving run status and results
  • Searching for packages and codebases
  • Triggering new runs
  • Managing merge proposals
  • Accessing build logs and artifacts

See the API documentation for full details and examples.
