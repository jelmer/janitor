What version control systems are supported?

The Janitor supports several version control systems and hosting platforms:

Version Control Systems

  • Git - Full support for Git repositories
  • Bazaar (bzr) - Legacy support for Bazaar repositories

Hosting Platforms

  • GitHub - Pull requests and direct pushes
  • GitLab - Merge requests and direct pushes (including self-hosted instances)
  • Launchpad - Merge proposals for Bazaar and Git
  • Salsa (Debian GitLab) - Specialized support for Debian packaging

Protocol Support

  • HTTPS repositories (public and private with authentication)
  • SSH repositories (with appropriate key configuration)
  • Git protocol (for public repositories)
Note: While Bazaar repositories are supported for legacy reasons, new development focuses primarily on Git repositories. If you have a Bazaar repository, consider migrating to Git for better long-term support.

For repositories on other platforms or using other version control systems, contact us to discuss adding support.
