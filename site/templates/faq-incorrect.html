What if the Janitor made an incorrect change?

If the Janitor made an incorrect change, please:

  1. Close or reject the merge proposal if it hasn't been merged yet
  2. Revert the change if it has already been merged
  3. Report the issue by filing a bug report

When reporting the issue, please include:

  • The repository where the incorrect change was made
  • A link to the merge proposal or commit
  • A description of what was wrong with the change
  • The expected behavior

We continuously improve the Janitor's detection and fix algorithms based on feedback. Your reports help make the system better for everyone.

Note: The Janitor is designed to be conservative and only make changes it's confident about. However, automation can never be perfect, and we appreciate your patience and feedback as we work to improve the system.
