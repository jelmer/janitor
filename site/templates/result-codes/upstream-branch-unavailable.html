Upstream Branch Unavailable

The upstream branch specified for this package could not be accessed. This prevents the Janitor from checking for new upstream versions or applying upstream fixes.

Common Causes:
  • Repository URL has changed or is no longer accessible
  • Branch name has changed or been removed
  • Network connectivity issues
  • Authentication problems for private repositories
  • Repository hosting service is down
Possible Solutions:
  • Verify the upstream repository URL is correct
  • Check if the branch name has changed
  • Update the Vcs-* fields in debian/control if needed
  • Contact upstream maintainers if the repository seems missing
