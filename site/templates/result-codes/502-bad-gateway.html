Bad Gateway

The server received an invalid response from an upstream server. This typically indicates a temporary service issue.

What this means:

A 502 Bad Gateway error occurs when a server acting as a gateway or proxy receives an invalid response from the upstream server it accessed to fulfill the request.

Typical Resolution:
  • This is usually a temporary issue - retry the operation
  • Check if the upstream service is experiencing issues
  • Wait a few minutes and try again
  • If the issue persists, it may indicate a service outage
