Worker Failure

An undefined error occurred while processing the package in the worker.

This is usually a bug in the Janitor.

What to do:
  • This error is typically not related to your package
  • The run may succeed if retried
  • If the error persists, please file a bug report
  • Include the run ID and any error details in your report
Escalation:

Worker failures should be investigated by the Janitor maintainers. Please file an issue with the run details.
