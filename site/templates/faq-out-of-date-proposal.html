What if a merge proposal is out of date?

If a merge proposal from the Janitor has become out of date (e.g., due to conflicts with changes you've made), you have several options:

  1. Wait for automatic refresh: The Janitor periodically refreshes merge proposals that have become out of date. This usually happens within a few days.
  2. Manually trigger a refresh: If you have access to the Janitor's web interface, you can manually trigger a refresh of the merge proposal.
  3. Close and reschedule: Close the merge proposal and reschedule the run for that package. The Janitor will create a new merge proposal with updated changes.
  4. Manual merge: If the conflicts are minor, you can manually resolve them and merge the proposal yourself.

The Janitor tracks the upstream branch and will automatically update proposals when possible. However, if there are conflicts that require human intervention, the proposal may need to be recreated.

Important: Don't force-push to branches created by the Janitor, as this can interfere with the automated update process.
